# Portfolio — Nguyen Duc Hanh

Single-page portfolio website for Unity Gameplay Developer profile.

## Open locally

Open `index.html` in a browser, or run a local server:

```powershell
cd d:\Projects\Portfolio\website
python -m http.server 8080
```

Then visit: http://localhost:8080

## Structure

```
website/
├── index.html
├── css/styles.css
├── js/main.js
```
